## 🎓 How to Use

### View the Implementation
```bash
# Open main dashboard
Navigate to: http://localhost:3000/dashboard

# Open unified dashboard
Navigate to: http://localhost:3000/dashboard/unified

# See empty states
(No action required - they show by default)

# Populate data
1. Add portfolio holdings
2. Make predictions
3. Charts update automatically
```

### Integrate in Your Page
```jsx
import { PortfolioChart } from '@/components/charts/PortfolioChart';
import { usePortfolio } from '@/context/PortfolioContext';

export default function MyPage() {
  const portfolio = usePortfolio();
  return <PortfolioChart holdings={portfolio?.holdings} />;
}
```

### Customize Empty States
```jsx
import { EmptyState } from '@/components/ui/EmptyState';

<EmptyState
  icon={CustomIcon}
  title="Custom Title"
  description="Custom description"
  variant="success"
/>
```
