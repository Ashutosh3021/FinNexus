================================================================================
                        KEY LEARNINGS
================================================================================

1. HYDRATION SAFETY IS CRITICAL
   Next.js/React requires data to match between server and client renders.
   Always use hydration guards when loading data from localStorage.

2. MOBILE-FIRST RESPONSIVE DESIGN
   Start with mobile styles, add breakpoints for larger screens.
   Prevents layout issues and ensures mobile UX.

3. SAFE DATA ACCESS PATTERNS
   Always use optional chaining (?.) and nullish coalescing (??)
   Prevents crashes from undefined data.

4. CONTEXT PROVIDER ORGANIZATION
   Wrap pages with multiple providers in layout.jsx, not individual pages.
   Ensures consistent state across entire app section.

5. RESPONSIVE SPACING AND TYPOGRAPHY
   Scale everything for mobile: padding, margins, font sizes, gaps.
   Improves readability and usability on all devices.
