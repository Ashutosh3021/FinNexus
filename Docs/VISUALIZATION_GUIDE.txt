5. USER FLOW - EMPTY TO POPULATED
═══════════════════════════════════════════════════════════════════════════════

New User Journey:

  1. USER LANDS ON DASHBOARD
     ↓
  2. SEES LOADING STATE
     └─ Spinner + "Loading..." message
     ↓
  3. PAGE LOADS - ALL EMPTY STATES
     └─ Portfolio Chart: "No holdings yet"
     └─ Performance Chart: "No data available"
     └─ Prediction Stats: "No predictions yet"
     ↓
  4. USER ADDS FIRST HOLDING
     ↓
  5. PORTFOLIO CHART UPDATES
     └─ Displays pie chart
     └─ Shows allocation
     ↓
  6. USER MAKES PREDICTIONS
     ↓
  7. PREDICTION STATS UPDATES
     └─ Displays bar chart
     └─ Shows accuracy
     ↓
  8. FULL DASHBOARD POPULATED
     └─ All charts displaying
     └─ Real-time updates enabled


6. VISUAL CUES FOR DATA STATES
═══════════════════════════════════════════════════════════════════════════════

LOADING STATE:
  Icon: Spinning circle (🔄)
  Color: Blue
  Animation: Rotating 360°
  Message: "Loading your data..."

EMPTY STATE:
  Icon: Chart or Alert icon (📊 or ⚠️)
  Color: Blue or Gray
  Animation: None (static)
  Message: "No [data type] yet"

DATA AVAILABLE:
  Icon: N/A (chart is visible)
  Color: Multi-colored (chart colors)
  Animation: Smooth entrance
  Message: N/A (chart speaks for itself)

ERROR STATE:
  Icon: Error circle (❌)
  Color: Red
  Animation: None (static)
  Message: "Unable to load data"
